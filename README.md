## 🚀 Key Features Breakdown ⚡ 🛡️

### 🔬 1. Forensic IMINT (Imagery Intelligence)
GReverse goes beyond basic image recognition. Our analysis pipeline includes:
*   **🛰️ Geolocation Analysis**: Identifying landmarks, architectural styles, vegetation, and infrastructure to pin a location on a map.
*   **🕒 Chronolocation**: Analyzing shadow direction, sun position, and weather conditions to determine the exact time of day and season.
*   **🔍 Micro-Detail Recognition**: Detecting license plates, serial numbers on equipment, specialized uniforms, insignia, and text within the scene.
*   **📀 EXIF Meta-Extraction**: Deep-dive into internal image headers (GPS coordinates, camera sensors, software edits, timestamps).

### 🧠 2. The AI Orbital Core (Multi-Model)
We leverage multiple "brains" to ensure zero blind spots:
*   **🌌 Kimi-k2.6 (Moonshot/NIM)**: The "Philosopher-Investigator." It handles complex reasoning and long-context analysis. If one trail goes cold, it utilizes its massive context window to cross-reference global data.
*   **⚡ Nemotron-4 VL 8B (NVIDIA)**: The "Tactical Analyst." A high-speed Vision-Language Model optimized for real-time visual parsing and object classification with extreme precision.

### 🛡️ 3. Legal Forensic Safeguards
We have implemented a **Legal Authorization Layer**. This protocol overrides generic AI safety refusals by verifying the investigative context as a 100% legal, authorized research mission. This ensures our agents never "censor" or "withhold" critical factual findings.

## 🏗️ Technical Architecture Deep-Dive 🏛️ 💻

### 🛰️ Front-End (The Tactical Dashboard)
*   **React + Next.js 15+**: Built on the bleeding edge of web performance.
*   **🎨 Tactical Dark Theme**: Using a high-contrast charcoal and cyan palette, optimized for professional environments.
*   **🖼️ Client-Side Optimization**: We implemented a custom **Canvas-based compression engine**. Images are resized and compressed on-the-fly to stay under strict token limits, preventing "400 Bad Request" errors during large forensic transfers.

### 🔌 Backend & API Intelligence
*   **🔒 Zero-Exposure Security**: All API keys are managed server-side. Your secrets never touch the client browser.
*   **🔄 Failover Intelligence**:
    *   **Primary Path**: NVIDIA NIM Integrate (V1).
    *   **Fallback Path**: Native Moonshot API Gateways.
    *   **Automatic Handshake**: The system automatically attempts both Vision and Text formats to ensure the report generates, no matter the input format.

## 📖 Operational Guide 🧭 🛰️

1. **Upload Phase**: Drop an image file or paste from the clipboard.
2. **Metadata Scan**: The UI immediately parses EXIF data for preliminary intel.
3. **Intelligence Request**: Input a custom investigative prompt (or use the default).
4. **Agent Selection**: Choose **Kimi-k2.6** for deep context reasoning or **Nemotron VL** for visual speed.
5. **Synthesis**: Watch as the AI builds a structured, professional forensic report in real-time.
6. **Documentation**: Export the markdown findings for your official dossier.
